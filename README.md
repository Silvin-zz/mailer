# Mailer
My first gem test

## Usage
It will be used to send data to mailer api micro-service :D.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'mailer', github: 'Silvin/mailer', branch: 'master'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install mailer
```
